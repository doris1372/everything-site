# Everything ✨

Экспериментальный одностраничный сайт — "всё в одном месте". Калькуляторы, игры,
генераторы, рисовалка, часы, таймеры, пианино и ещё куча всего. Работает целиком
в браузере, без бэкенда.

## 🚀 Демо

После того как PR с GitHub Pages workflow смержится в `main`, сайт будет доступен
по адресу: **https://doris1372.github.io/everything-site/**

## 📦 Структура

```
everything-site/
├── index.html          # Точка входа
├── css/style.css       # Стили + темы (тёмная/светлая)
├── js/
│   ├── app.js          # Навигация, роутинг, поиск, тема
│   └── sections.js     # Все модули (игры, утилиты, генераторы)
└── .github/workflows/
    └── pages.yml       # Автодеплой на GitHub Pages
```

## 🧩 Разделы

**Утилиты:** калькулятор, конвертер единиц, Base64, JSON форматтер,
текст-инструменты, Markdown превью, BMI, калькулятор возраста, секундомер,
таймер, Pomodoro, заметки, Todo, эмодзи, генератор паролей.

**Генераторы:** случайное число, Lorem ipsum, цвет + палитра, CSS градиент,
никнеймы.

**Игры:** крестики-нолики (с непобедимым ботом — minimax), змейка, угадай число,
камень–ножницы–бумага, память, кликер.

**Развлечения:** кубик, монета, колесо фортуны, цитаты, анекдоты, рисовалка,
фейерверки, чат-бот.

**Звук:** пианино (Web Audio), метроном.

**Инфо:** аналоговые + цифровые часы, календарь.

## 🛠 Локальный запуск

Никакой сборки не нужно — это чистый HTML/CSS/JS. Достаточно любого статического
сервера:

```bash
python3 -m http.server 8000
# или
npx serve .
```

Затем открыть `http://localhost:8000`.

## 🧑‍💻 Добавить новый раздел

Откройте `js/sections.js` и добавьте новый `reg({...})`:

```js
reg({
  id: "my-section",
  title: "Мой раздел",
  icon: "🌟",
  group: "Утилиты", // или Игры, Генераторы, Развлечения, Звук, Инфо
  desc: "Краткое описание",
  render(root, { toast, go }) {
    root.appendChild(/* ваш DOM */);
  },
});
```

Если раздел создаёт таймеры / слушателей `document`, очистите их через
`root._cleanup = () => { /* unsubscribe */ }` — роутер вызовет при переходе.
